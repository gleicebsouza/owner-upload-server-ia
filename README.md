### Executing the back-end
You need to have Node.js / NPM installed locally.

1. Install all the required dependencies:

```
npm install
```

2. Execute the project:

```
npm run dev
```

3. Execute prisma:

```
npx prisma studio
```

Prisma Studio is up on http://localhost:5555
